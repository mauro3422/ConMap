## ✨ ¿Qué es ConMap?

**ConMap** (Concept Map) no es solo un visualizador; es un **ecosistema dinámico de aprendizaje**. Diseñado como un "cuaderno inteligente", permite a estudiantes y profesionales organizar conceptos complejos mediante diagramas vectoriales de alta precisión, garantizando una navegación fluida y una arquitectura totalmente modular.

### 🚀 Capacidades Core

*   **🎨 Leyendas Dinámicas Universales**: El sistema se adapta a tu materia.
*   **➕ Creador de Mapas Estructurados**: Genera esquemas complejos en segundos.
*   **🔄 Sincronización Multi-Pestaña**: Cambios en tiempo real entre múltiples ventanas (Live Sync).
*   **💾 Persistencia Local (Offline)**: Todo tu trabajo se guarda en tu navegador.
*   **🛡️ Gestión de Ciclo de Vida**: Control total (CRUD) sobre tus diagramas con notificaciones instantáneas.

---

## 🛠️ Guía de Uso Rápido

1.  **Exploración**: Usa el ícono del **Libro** 📓 en la barra superior para navegar entre tus temas guardados.
2.  **Creación**: Haz clic en **"+ Nuevo Mapa"** para abrir el taller de diseño. Define tus categorías (ej: *Teoría, Práctica, Casos*) y tus ramas iniciales.
3.  **Interacción**: 
    *   **Click + Arrastrar**: Mueve el lienzo con precisión quirúrgica.
    *   **Scroll / Botones**: Haz zoom para profundizar en los detalles.
    *   **Modo Interactivo**: Activa la lupa para una experiencia de visualización enfocada.
4.  **Gestión**: Pulsa la **Papelera Roja** 🗑️ a la izquierda de cualquier tema personalizado para eliminarlo al instante.

## 🏗️ Arquitectura Técnica

El sistema ha sido construido bajo estándares de **alto rendimiento** y **código limpio**:

*   **Frontend**: HTML5 Semántico & CSS3 con variables dinámicas y Glassmorphism.
*   **Engine**: Motor de renderizado SVG nativo para escalabilidad infinita.
*   **State Management**: `MapState` reactivo que sincroniza metadatos y topología.
*   **Persistence Layer**: Driver de `LocalStorage` con manejo de colisiones y snapshots dinámicos.
